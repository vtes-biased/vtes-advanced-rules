# VTES Advanced Rules

[![Read online](https://img.shields.io/badge/Read%20online-VTES%20Advanced%20Rules-8a1f1f?style=for-the-badge)](https://vtes-biased.github.io/vtes-advanced-rules/)
[![GitHub Pages](https://img.shields.io/github/deployments/vtes-biased/vtes-advanced-rules/github-pages?label=github%20pages)](https://vtes-biased.github.io/vtes-advanced-rules/)

A judge-level companion to the [VTES rulebook](https://www.vekn.net/rulebook),
synthesizing thirty years of rulings into general principles organized by game
mechanic.

**Read it online → <https://vtes-biased.github.io/vtes-advanced-rules/>**

The document itself is [`docs/advanced-rules.md`](docs/advanced-rules.md); the
GitHub Pages site is built from the [`docs/`](docs/) directory.
